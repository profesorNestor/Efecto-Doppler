# 🌊 Cuaderno Interactivo: Efecto Doppler

[![Ver Demostración](https://img.shields.io/badge/Ver-Demostración_en_Vivo-blue?style=for-the-badge)](https://profesornestor.github.io/Efecto-Doppler/)

Este repositorio aloja un cuaderno virtual interactivo diseñado para facilitar la enseñanza y el aprendizaje del **Efecto Doppler**. A través de la experimentación directa, esta herramienta permite visualizar y comprender cómo el movimiento relativo entre una fuente de ondas y un observador altera la frecuencia que este último percibe.

## 🎯 Objetivo Educativo
Brindar a los estudiantes una plataforma interactiva donde los conceptos abstractos de la física se vuelvan tangibles, permitiéndoles manipular las variables cinemáticas del sistema y observar el comportamiento de las ondas de forma instantánea.

## ✨ Características Principales
* **Simulaciones en Tiempo Real:** Visualización dinámica de la compresión y expansión de los frentes de onda.
* **Interactividad Paramétrica:** Controles deslizantes e inputs para ajustar de forma intuitiva la velocidad de la fuente emisora, la velocidad del observador y la frecuencia original.
* **Cálculos Físicos:** Muestra matemáticamente los cambios en la frecuencia aparente según las variables introducidas por el usuario.
* **Diseño UI/UX:** Interfaz limpia, amigable y responsiva, pensada para mantener la concentración del estudiante.

## 🛠️ Tecnologías Utilizadas
Este proyecto está desarrollado con tecnologías web nativas para asegurar su ejecución rápida y sin dependencias en cualquier navegador moderno:
* **HTML5:** Estructura semántica del cuaderno interactivo.
* **CSS3:** Estilos de la interfaz gráfica del simulador.
* **JavaScript:** Lógica principal, motor de la simulación física, renderizado y cálculos matemáticos interactivos.
